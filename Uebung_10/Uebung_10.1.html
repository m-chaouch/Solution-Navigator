10.1 Static classes in TypeScript

Gibt es in TypeScript statische Klassen? Erläutern Sie den Unterschied von TypeScript zu anderen objekt-orientierten, statisch typisierten Programmiersprachen wie Java und C#.




In Typescript gibt es direkt keine statischen Klassen, aber es gibt mehrere Wege, mit denen man eine statische Klasse simulieren kann. Beispielsweise den Konstruktor einer Klasse privat machen. Unterschiede:

  • Viele Unterschiede in der Syntax
  • Kompilierung des Quellcodes ist anders. In Java wird der Quellcode in Bytecode kompiliert und dann von der JVM (Java Virtual Machine) interpretiert oder von der JIT (Just in. TypeScript wird erstmal zu JavaScript-Code kompiliert und dann in der JavaScript runtime ausgeführt. Deshalb hat man nach der Kompilierung des TypeScript-Quellcodes eine JavaScript-Datei.
  • Da TypeScript sehr ähnlich zu JavaScript ist und sowieso zu JavaScript Code kompiliert wird, ist TypeScript 100% mit JavaScript kompatibel, anders als Java oder C#.
