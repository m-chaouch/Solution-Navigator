3.1 Responsiv mit Flexbox und Desktop-First

Spielen Sie zunächst das Flexbox Froggy-Spiel, um Flexbox zu lernen. Implementieren Sie dann ausschließlich mit HTML, CSS und Flexbox das folgende Responsive Webdesign nach der Desktop-First-Strategie:





HTML Code: 

                        
                    <!DOCTYPE html>

                    <html lang="en">

                    <head>

                        <meta charset="UTF-8">

                        <meta name="viewport" content="width=device-width, initial-scale=1.0">

                        <title>Übung 3.1</title>

                    </head>

                    <body>

                        <div class="bigbox">

                            <div class="redbox"></div>

                            <div class="greenbox"></div>

                            <div class="bluebox"></div>

                            <div class="pinkbox"></div>

                        </div>

                    </body>

                    </html>
