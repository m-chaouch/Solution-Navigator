Spread-Operator

Schreiben Sie einen kurzen Erklärartikel zu dem „Spread“-Operator mit einem Beispiel.



Der ... Operator dient je nach dem wie er notiert ist, als Rest-Operator oder als Spread-Operator. Der Spread-Operator wird nur angewendet, wenn dem ... Operator ein iterierbares Objekt übergeben wird, während sich diese in einem Array oder in einem Funktionsaufruf befinden. Dann fügt der Spread-Operator die Elemente des Objektes in ein Array ein oder im Falle des Funktionsaufrufes übergibt dieser die einzelnen Elemente des Objektes als Argumente an die Funktion.

Beispiel 1:


            
            const numbers = [1, 2, 3, 4];
            const numbers2 = [5, 6, 7, 8];


            const spread1 = [...numbers, ...numbers2];

            console.log(spread1);

Ausgabe: [1, 2, 3, 4, 5 ,6 ,7 ,8]

Zu Beachten:

Was ist wenn der Array zu viele Argumente hat?

            
                const numbers = [1, 2, 3, 4];

                function ausgabe(a, b ,c) {
                    return [a, b, c];
                }
                    
                console.log(ausgabe(...numbers));

Ausgabe: [1, 2, 3] -> Das letzte Element wird nicht eingesetzt.



Was ist wenn der Array zu wenige Argumente hat?

            
                const numbers = [1, 2];

                function ausgabe(a, b ,c) {
                return [a, b, c];
                }

                console.log(ausgabe(...numbers));

Ausgabe: [1, 2, undefined] -> Mit einem Element zu wenig ist das letzte Element im Array undefined.
